# Tarjimly

Tarjimly is a simple yet powerful CLI translator build using MyMemory translated API

## What's inside

```bash
├── bin
│   ├── tarjimly.js
├── index.js
├── package-lock.json
├── package.json
└── readme.md
```

## Install

```bash
npm i tarjimly -g
```

## Run

```bash
tarjimly <word> <from language> <to language>
```

## Example

```bash
tarjimly "car" en ar
```
